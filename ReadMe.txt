The Code folder has three sub folders:
1)Supriya_CrimeAnalysis -> This has the map reduce program that uses the first characters of northern and eastern region codes along with crime types as the key

2)Supriya_CrimeAnalysis3-> This has the map reduce program that uses all the characters of northern and eastern region codes along with crime types as the key.

3)Supriya_CrimeAnalysis2 -> This has the map reduce program that uses the first three characters of northern and eastern region codes along with crime types as the key
